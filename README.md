Selenium Cucumber TestNG
----------------------------------------

[Selenium](https://www.selenium.dev/) along with  [Cucumber](https://cucumber.io/) and [TestNG](https://testng.org/doc/) to validate a web site.
 

#### To run the project

```bash
mvn test
```
